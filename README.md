# 🏴‍☠️ trove

Arbitrary JSON-structured data, all-indexing storage backed with [Sophia](https://github.com/pmwkaa/sophia)

## Installation

1. Add the dependency to your `shard.yml`:

   ```yaml
   dependencies:
     trove:
       github: mentalblood0/trove
   ```

2. Run `shards install`
